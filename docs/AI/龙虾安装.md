### 安装 OpenClaw 时报错 “gateway connect failed: Error: pairing required”，如何处理？

执行以下命令删除设备和身份，然后重新安装网关并重启解决该问题。

```
openclaw gateway install --force
openclaw gateway start
一直无法启动可以使用
openclaw gateway --port 18789 --verbose
```



##### 界面配置json 需要修改的配置信息如下：主要是中转站的key使用这个配置

- <ARK_API_KEY>：替换为自己的 [API Key](https://console.volcengine.com/ark/region:ark+cn-beijing/apikey)。

- models：更新为需要使用的模型信息。支持的模型信息参见[核心配置]()。

```shell
"auth": {
    "profiles": {
      "volcengine:default": {
        "provider": "volcengine",
        "mode": "api_key"
      }
    }
  },

"models": {
    "providers": {
      "volcengine": {
        "baseUrl": "https://ark.cn-beijing.volces.com/api/coding/v3",
        "apiKey": "你的key", 
        "api": "openai-completions",
        "models": [
          {
            "id": "glm-4.7",
            "name": "glm-4.7",
            "reasoning": false,
            "input": [
              "text"
            ],
            "cost": {
              "input": 0,
              "output": 0,
              "cacheRead": 0,
              "cacheWrite": 0
            },
            "contextWindow": 200000,
            "maxTokens": 8192
          }
        ]
      }
    }
 }
```

##### 飞书接入

1. 进入飞书开发者平台

2. 创建应用 -> 添加机器人

3. 开通权限 直接导入openclaw 提供的json

  ```json
  {
    "scopes": {
      "tenant": [
        "aily:file:read",
        "aily:file:write",
        "application:application.app_message_stats.overview:readonly",
        "application:application:self_manage",
        "application:bot.menu:write",
        "cardkit:card:write",
        "contact:user.employee_id:readonly",
        "corehr:file:download",
        "docs:document.content:read",
        "event:ip_list",
        "im:chat",
        "im:chat.access_event.bot_p2p_chat:read",
        "im:chat.members:bot_access",
        "im:message",
        "im:message.group_at_msg:readonly",
        "im:message.group_msg",
        "im:message.p2p_msg:readonly",
        "im:message:readonly",
        "im:message:send_as_bot",
        "im:resource",
        "sheets:spreadsheet",
        "wiki:wiki:readonly"
      ],
      "user": ["aily:file:read", "aily:file:write", "im:chat.access_event.bot_p2p_chat:read"]
    }
  }
  ```

4. 去凭证与基础信息里面将 appid secret复制下来

5. 创建版本信息(版本管理与发布里面)

6. 去安装openclaw的机器执行 openclaw channels add

  ```shell
  1 选择 yes
  2 选择 飞书
  3 选择 Download from npm (会报错,手动安装 去openclaw安装目录下运行 npm i @openclaw/feishu) 
  4 选择 本地插件
  5 输入 appid与密钥
  6 选择feishu-china
  7 选择allowlist - only respond in specific groups
  8 输入 白名单会话id,多个之间逗号隔开
  9 看到Configure DM access policies now? (default: pairing) 直接使用默认的选项 使用配对模式
  10 看到Add display names for these accounts? (optional) 也是默认选项 不起名字
  11 重启网关 openclaw gateway restart
  12 查看插件列表,确认飞书插件已启用 openclaw plugins list
  13 如果没有启用 手动启用 openclaw plugins enable feishu
  ```

  

7. 设置飞书事件与回调  ***必须先按照官方的文档设置Channel,不然无法保存设置***

8. 订阅方式改为长链接, 然后添加事件, 搜索 接收消息 , 勾选添加即可

9. 给机器人发消息, 机器人会回复一串配对信息

  ```tex
  OpenClaw: access not configured.
  Your Feishu user id: ou_b1dcf0d8132dfcfeffc001688cbae20c
  Pairing code: S3N8LLCC
  Ask the bot owner to approve with:
  openclaw pairing approve feishu S3N8LLCC
  直接在电脑上运行最后一段即可完成配对, 完成所有配置
  ```
